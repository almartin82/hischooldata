# Download DBEDT State Data Book enrollment data (2010-2017)

Downloads enrollment data from the Hawaii DBEDT State Data Book. Files
are Excel tables with state-level and county-level enrollment.

## Usage

``` r
download_dbedt_enrollment(end_year)
```

## Arguments

- end_year:

  School year end (2010-2017)

## Value

Data frame with enrollment data
